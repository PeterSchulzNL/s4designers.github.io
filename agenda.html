Friday Jan 28th:

    Do you have some knowlegde of HTML and CSS? These are computer languages for creating web pages. They are not programming languages; HTML describes the content and structure of a web page, and CSS is used to describe the colors, fonts, sizes and other visual design properties of the elements in the HTML file.

    Because you can't write computer programs in HTML/CSS, this course is not going to invest time teaching them. But to be able to use JavaScript for interesting things, a little bit of very basic knowledge about HTML and CSS is absolutely necessary.

    If you're unsure about your HTML/CSS knowlegde, check out the online course on the Kahn Academy on HTML and CSS: Intro to HTML/CSS: Making webpages

    For this S4D course, you only need the first four chapters of their course:

    1. Intro to HTML
    2. Intro to CSS
    3. More HTML tags
    4. CSS text properties
    The rest of the course is interesting too, but not needed for this programming course.
